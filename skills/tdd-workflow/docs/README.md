# 🧪 TDD Workflow (Free Edition)

Welcome to the TDD Workflow module!

**Purpose:** This skill introduces Test-Driven Development to your project. It configures a basic testing environment (Jest) and encourages the AI to write tests before application code, reducing bugs.

## How to use:
> *"Use the `tdd-workflow` skill. Setup tests for my math functions."*

---
> 💎 **Passez au niveau supérieur :** Ce skill gratuit est génial pour s'initier aux tests. Mais pour une application stricte en production (qui empêche physiquement l'IA de coder sans tests), découvrez le skill `strict-tdd-enforcer` de notre **Pack Premium Antigravity**.
