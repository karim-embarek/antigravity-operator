# 🇩🇿 Law 18-07 Auditor (Free Edition)

Welcome to the Algerian Compliance module!

**Purpose:** La loi 18-07 encadre strictement la collecte de données en Algérie. Ce skill analyse votre code et votre architecture pour s'assurer que vous respectez les règles de l'ANPDP (Autorité Nationale de Protection des Données à Caractère Personnel), notamment la localisation des données.

## How to use:
> *"Utilise le skill `law-18-07-auditor` pour vérifier si mon architecture SaaS est légale en Algérie."*
