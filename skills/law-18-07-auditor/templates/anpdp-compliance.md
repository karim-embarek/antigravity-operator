# ANPDP Compliance Checklist (Loi 18-07)

## 1. Localisation des Données (Article 44)
- [ ] La base de données est-elle hébergée physiquement sur le territoire algérien (ex: Icosnet, Djaweb, ISSAL) ?
- [ ] Y a-t-il des transferts de données vers des serveurs étrangers sans l'accord explicite de l'ANPDP ?

## 2. Consentement Explicite (Article 7)
- [ ] L'utilisateur doit cocher une case (non pré-cochée) pour accepter le traitement de ses données.
- [ ] Le consentement est-il tracé et horodaté dans la base de données ?

## 3. Sécurité et Confidentialité (Article 41)
- [ ] Les mots de passe sont-ils hachés (ex: bcrypt/argon2) ?
- [ ] Les communications sont-elles chiffrées (HTTPS/TLS 1.2 minimum) ?
