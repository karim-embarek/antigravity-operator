# 🗄️ Database Optimizer (Free Edition)

Welcome to the Database Performance module!

**Purpose:** This skill turns your AI into a Senior DBA. It will scan your Prisma schemas and your SQL queries to detect N+1 query problems, missing indexes, and relations that will slow down your app in production.

## How to use:
> *"Utilise le skill `database-optimizer` pour auditer mon fichier `schema.prisma` et mes routes d'API."*
