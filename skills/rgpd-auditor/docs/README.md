# 🇪🇺 RGPD Auditor (Free Edition)

Welcome to the European Compliance module!

**Purpose:** This skill ensures your SaaS doesn't violate European GDPR rules. The agent will scan your database schemas to ensure you aren't storing sensitive data illegally and will generate a Privacy Policy template for you.

## How to use:
> *"Utilise le skill `rgpd-auditor` pour vérifier mon fichier `schema.prisma` et générer ma politique de confidentialité."*
